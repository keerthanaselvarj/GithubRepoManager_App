# GiTHubRepoManager_App


GitHubRepoManager_App application to search and display Github information using Username  or by Technology
.



Global dependencies:

node

mongodb

Express
gulp

Github URL:https://github.com/stackroute-hybrid/GiTHubRepoManager_App


To run the application:


1. Clone the repo

2. From GiTHubRepoManager_App/client run npm install

3. From GiTHubRepoManager_App/server run npm install

4. From GiTHubRepoManager_App/client run gulp

5. From GiTHubRepoManager_App/server run npm start
